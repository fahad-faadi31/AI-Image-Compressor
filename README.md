# AI Image Compressor

A production-ready AI-powered image compression application built using PyTorch, FastAPI, React, and Tailwind CSS.

##  Features (Planned)

- Deep Learning-based Image Compression
- FastAPI Backend
- React + Tailwind CSS Frontend
- Drag & Drop Upload
- Image Comparison Slider
- PSNR & SSIM Metrics
- Compression Ratio Analysis
- Download Compressed Images
- Batch Image Compression
- Responsive Modern UI

##  Tech Stack

- PyTorch
- FastAPI
- React
- Tailwind CSS
- Framer Motion
- OpenCV
- Pillow

 ##  Project Status

 Under Active Development

Each commit represents a working milestone in the development process.
